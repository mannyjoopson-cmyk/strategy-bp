# Security And Privacy

## Supported Version

Security and privacy fixes are accepted for the latest released version.

## Reporting

Use a private GitHub security advisory when the repository enables that feature. Do not post credentials, personal data, private research materials, or exploit details in a public issue.

## Data Boundary

The plugin ships no credentials, telemetry, hooks, MCP servers, automatic outreach, or data-purchasing integration. Users are responsible for approving and configuring any external tools or data access used with Codex.
